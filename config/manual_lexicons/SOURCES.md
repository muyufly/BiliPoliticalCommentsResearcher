人工词库来源说明

本词库是研究性操作定义，不等同于平台官方分类或稳定学术分类。

本次预填主要参考公开可访问材料中的用法线索：

- 维基百科：中国大陆网络用语列表、中国新左派、中国自由派、改革开放、文化大革命、反右运动、封建制度、中国古代政治史、宗法等条目。
- 公开百科与文献条目：《一九八四》、真理部、老大哥、王洪文、四人帮、皇汉、毛泽东诗词、改革开放等条目。
- 党史与公开史料：毛泽东诗词中的“换了人间”“萧瑟秋风今又是”“四海翻腾云水怒”等意象，以及“人民万岁”“世界人民大团结万岁”“联合起来”等政治标语。
- 歌词互文校正：《四海五洲》公开歌词中“囿于四海和五洲之内”“让翻腾的风雷”等拆分表达，按毛泽东《满江红·和郭沫若同志》“四海翻腾云水怒，五洲震荡风雷激”的变体引用处理；“克里姆林宫的花”“褪色的口号”“文娱汇演集体活动”等作为社会主义阵营、政治标语记忆和群众文艺动员的辅助证据。
- 谐音/近形表达：参考中文网络谐音词与拼音相似度研究，把同音、近形、拆字、变体写法作为“低置信补充线索”，必须保留置信度与人工复核标记。
- Bilibili 公开专栏：古代政治制度、宗法制、郡县制、中央集权等历史制度梳理。
- Zhihu / Bilibili / YouTube 搜索结果：用于观察“神友、兔友、皇汉、网左、乐子人”等标签在公共讨论中的共现与语境。
- 贴吧公开页采样：神系来源与迁移主要参考“神奈川冲浪里吧”公开讨论与“新家是孙吧”等帖文；但按项目内人工校正，`孙笑川吧 / 孙吧` 本身不直接判为 `神`，只作为抽象/乐子低权重线索。
- 贴吧公开页采样：`朴正熙吧`、`中国人口吧` 常直接使用“神 / 兔 / 口 / 航”等元标签讨论阵营成分，可作为观察自称/互称的公共样本，但不把 `朴吧 / 口吧` 本身硬编码为单一立场。
- 贴吧公开页采样：`那年那兔那些事儿吧` 与 `航空母舰吧` 常被并称为兔友/航吧语境，`种花家 / 大国梦 / 星辰大海 / 一盘大棋` 等词按兔系国家叙事处理。
- 贴吧公开页采样：`皇汉 / 黄汉 / 汉皇 / 明粉 / 满清 / 反清复明 / 朱元璋` 等词参考 `皇汉吧 / 汉皇吧 / 黄汉笑话吧` 及相关公开讨论页补充，用作皇系民族主义与明粉叙事线索。
- 贴吧公开页采样：`网左吧` 与相关左翼讨论页中可见 `旺座 / 康米 / 实左 / 少徒 / 远视主义` 等网络左翼自称，也可见“古风DJ”“王座/旺座”与左翼传播语境的结合，用作左派圈层黑话补充。
- 知乎公开内容采样：围绕《毛泽东选集》、`教员`、`阶级史观`、`历史唯物主义`、`修正主义`、`无产阶级专政`、`资本主义复辟` 的公开问答、文章与专题，可作为理论学习型左翼词条与反修话语的补充来源。
- 操作细化：左派词库额外区分 `网络左派`、`马列毛理论学习`、`阶级分析与人民史观`、`反修防修与资本主义复辟`、`劳工剥削与阶层批判`、`社会主义阵营怀旧` 六类，避免把所有左派词都压进单一“大左派”桶。
- 贴吧公开页采样：`航空母舰吧` 等公开讨论中常见 `大国重器 / 东风快递 / 国运 / 遥遥领先` 一类强军科技强国表达，可作为兔系中“强军科技强国叙事”的补充线索；`种花家 / 我兔 / 一盘大棋` 则更适合作为国家拟人与大战略叙事。
- 贴吧与知乎公开页采样：`神奈川冲浪里 / 冲浪里 / 神奈冲 / 何意味` 可作为神系圈层黑话线索；`1984 / 真理部 / 大洋国` 与 `润学` 更适合作为神系中的“奥威尔式威权批判”“移民逃离叙事”。
- 贴吧与知乎公开页采样：`尊王攘夷 / 华夷之辨 / 汉统 / 汉贼不两立 / 再造华夏 / 朱重八` 等词更适合细分到皇系中的 `华夷之辨与汉本位`、`明粉复明叙事`、`明清易代创伤叙事`，而不是全部并入单一 `皇汉民族主义`。
- Discord：多数相关讨论位于私域服务器或登录墙内，本项目不采集私密内容，只保留公开搜索能验证的弱线索。

使用原则：

- `semantic_overrides.json` 只处理“词条是否具有政治/历史含义，以及归入哪个真正含义”。
- `stance_overrides.json` 只处理“词条或真正含义对 神/左/兔/皇/乐子人 的加权倾向”。
- 有强反讽、引用他人观点、否定句时，仍需结合 AI 复核或人工复核。
- 拆分互文优先写入 `composite_overrides.json`，不要把所有碎片词都强行升为高权重单词，避免误伤普通歌词意象。
- AI 纠偏建议先进入 `lexicon_candidates.json`，只有人工确认后才合并到正式词库。
- “小王”“年轻人”“小崔”“毛”“猫”“稻”“司马懿”等代称具有强歧义性。本词库把它们作为高权重研究线索，而不是身份确认；报告中的证据评论仍应人工抽查。
- 古风/古典文学元素采用“语境触发”原则：“香草美人”“屈原”“离骚”“芳草”“美人迟暮”等会被标记为古典文学政治寄托，但实际立场仍依赖上下文和 AI/人工复核。
